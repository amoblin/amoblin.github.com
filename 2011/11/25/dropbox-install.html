国内Dropbox安装使用

home tags links feed

目前Dropboxd的情况是http不能访问，https可以访问，和google docs,google reader一样。

所以下面是讲如何在https下完成安装。

下载在线安装包：

$ wget https://www.dropbox.com/downloading?os=lnx

os的值可以是win,lnx,mac。

下载包很小，安装也很快，但注意，这只是安装了一个空壳，接下来才是真正的安装：

$ dropbox start -i

这样会启动安装程序，自动从官网下载数据包进行在线安装。

但是我们访问不了，所以安装失败。怎么办呢？手工下载吧。

下载数据包到主目录，目前最新版是1.2.48：

$ cd ~
$ wget https://dl-web.dropbox.com/u/17/dropbox-lnx.x86-1.2.48.tar.gz

修改dropbox脚本，替换网上下载数据包为使用本地数据包：

$ sudo sed -i 's/urlopen(.*)/urlopen(os.path.join(os.path.expanduser("~"), "dropbox-%s-%s.tar.gz" % (plat(), "1.2.48")))/' `which dropbox`

然后就可以安装啦:

$ dropbox start -i

查看运行状态:

$ dropbox status
Downloading 909 files (5.9 KB/sec, 2 days left)

获取public url：

$ dropbox public filename
