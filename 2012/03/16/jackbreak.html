越狱前准备工作：

  1. 下载安装tiny umbrella和redsn0w。

1.备份shsh。使用小雨伞或红雪备份。

2.备份数据：照片，通讯录，短信等。连接itunes，右键iphone设备，点“备份”。

  1. 了解iphone状态：

    DFU模式：Power键3秒；接着不松开Power，同时按Home键，大约15秒；松开Power，继续按着Home键，大约15秒。 恢复模式：

DFU模式 如果你的手机固件已经出现问题需重刷，并且itunes也无法识别iphone,不用急，你只要同时按下Power + Home按键，保持10秒钟左右 当屏幕黑了以后 松开Power键，但不要松开Home键，这时候屏幕仍然是黑的，启动iTunes（别松开Home键），iTunes会检测到iPhone处于恢复模式（出现提示窗口），现在可以松开Home键了。这时按着SHIFT键点ITUNES的恢复，选择你所下载的固件版本。和上面步骤就一样了。"

iphone无法开机了，怎么办？

假死现象很常见，也很好解决。

按住HOME+POWER 20s之后就可以重启了

在固件刷完之后，如果你想要回当前设备的备份请选择恢复备份，不要选择设置为一个新的设备。

iOS中Unix环境配置

使用ssh登陆系统

$ ssh root@192.168.2.2

初始密码为alpine，最好修改一下。

添加178源

# echo "deb http://apt.178.com/ ./" >> /etc/apt/sources.list.d/cydia.list
# apt-get update
  1. 安装aptitude：
# apt-get install aptitude
  1. 使用aptitude搜索软件包：
# aptitude search zephyr

Linux/Unix用户必备工具：

# apt-get install sudo less vim git python screen zsh

iPhone必备软件(Zephyr, NcSettings, KuaiDial)：

# apt-get install zephyr.178 com.jamied360.ncsettings kuaidial-beta.178

开发工具集：

aptitude adv-cmds basic-cmds bootstrap-cmds coreutils developer-cmds file-cmds network-cmds shell-cmds system-cmds inetutils

通过命令行发短信：

$ sudo apt-get install sendsms

iOS中常用命令

  1. 重启SpringBoard
$ sudo sbreload
  1. 重启iphone
$ sudo reboot
