Bash 有个功能叫大括号扩展，大括号包围的，用逗号隔开的参数会扩展为独立的多个参数。

   $ mv a{,.html}

   $ mv a.{html,htm}

   $ yum install git
   You need to be root to perform this command.
   $ sudo !!
   sudo yum install git
   ...

   $ mv /etc/apt/sources.{list,bak}



!* 将代替上一个命令的所有参数，(!! 是整条命令和所有参数)

!$ 上一条命令的最后一个参数

!:3 上一条命令的第3个参数
